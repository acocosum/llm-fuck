# LLM-Fuck: 大语言模型版的命令修正工具

LLM-Fuck 是一个智能的命令行修正工具，类似于著名的 "The Fuck"，但使用大语言模型来提供更智能的修正建议。当您在终端中输入错误命令时，只需执行 `fuck` 命令，LLM-Fuck 将分析错误并提供修正建议。

## 特性

- 🧠 使用大语言模型分析命令错误
- 🛠️ 支持多种常见命令错误修正
- 🌐 支持多种LLM提供商（OpenAI, Anthropic, 百度, 讯飞等）
- 🔄 提供上下文感知的修正建议
- 🚀 简单易用的界面和快捷命令
